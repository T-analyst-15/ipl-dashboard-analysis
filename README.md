# IPL 2024 Performance Dashboard

## Overview
An interactive dashboard analysing IPL 2024 match data to uncover 
team performance trends, player statistics, and win/loss patterns 
across the season.

## Tools Used
- Power BI / Tableau — dashboard and visualisations
- Python (Pandas) / Excel — data cleaning and preparation
- Dataset source: Kaggle / ESPN Cricinfo
